# Help Desk Support Project

This repository contains my final project for the PC Help Desk Support course.

## Repository Structure

- documentation/ — Troubleshooting guide, KB article, communication samples
- scripts/ — PowerShell, Python, or batch scripts (optional but encouraged)
- ai-transcripts/ — ChatGPT or Claude conversations, prompts, and AI usage summary
- assets/ — Images, screenshots, logs, exported reports

## Tools Used
- GitHub
- Visual Studio Code
- AI assistant (ChatGPT or Claude)

## Purpose
This project demonstrates help desk skills such as troubleshooting, documentation, workflow organization, and basic automation.

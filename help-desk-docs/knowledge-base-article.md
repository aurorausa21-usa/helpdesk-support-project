## Purpose
this article explains how to reset the windows network adapter when experiencing internet issues.

## Steps
1. open **settings**  
2. go to **network & internet**  
3. click **advanced network settings**  
4. select **network reset**  
5. click **reset now**  
6. restart the computer

## When to Use
- slow wi-fi  
- frequent disconnections  
- cannot obtain ip address  
- adapter failure after update

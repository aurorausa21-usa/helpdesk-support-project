## Symptoms
- User reports no internet access
- Wi-Fi icon shows disconnected
- Other devices can connect normally

## Step-by-Step Troubleshooting

1. Confirm Airplane Mode is off
2. Restart Wi-Fi adapter
3. Run Windows Network Troubleshooter
4. Forget and reconnect to the network
5. Verify correct password
6. Restart router if needed
7. Check IP configuration
8. Reinstall network adapter drivers

## Resolution

Issue resolved after resetting the network adapter and reconnecting to the correct SSID.
